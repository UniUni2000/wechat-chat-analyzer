# 微信聊天深度分析 — LLM 分析指南

本文件是你 (LLM) 进行深度分析时的参考手册。当 SKILL.md 中的 Step 4 要求你生成分析报告时，参照此指南。

## 1. "一语胜千言" 回忆杀金句

### 创作原则
- **一句话**浓缩整段关系的精髓
- 要有**画面感**、**网感**、**文艺气息**
- 既能打动当事人，也能让旁人会心一笑
- 根据关系类型调整风格

### 关系类型判断依据

| 关系类型 | 信号特征 |
|---|---|
| **情侣/暧昧** | 高频"宝贝/亲爱的/想你"、晚安消息、高互动密度、深夜聊天多 |
| **死党/闺蜜** | "哈哈哈"占比高、互怼频繁、分享日常琐事、表情包大战 |
| **同事** | 工作相关词频高、聊天集中在工作时间、消息偏正式 |
| **暧昧期** | 一方主动明显更多、回复时差不对称、试探性问句多 |
| **家人** | 关心身体/吃饭、消息简短直接、固定时段联系 |
| **普通朋友** | 聊天频率低、集中在特定话题、互动模式平衡 |

### 金句风格示例 (仅供参考，请创造独特的)

**情侣型**:
- "从第一句'在吗'到每晚的'晚安'，{days}天的碎碎念，原来都是情书。"
- "你的消息提示音，是我这一整年最好听的BGM。"

**死党型**:
- "我们聊了{count}条消息，其中{pct}%是哈哈哈。剩下的，是不能告诉别人的秘密。"
- "{count}条消息、{days}个日夜，我们把'无聊'聊成了'无价'。"

**暧昧型**:
- "你发来的{count}条消息里，我数了{init}次你主动找我——每一次，心跳都快了0.5秒。"
- "凌晨{hour}点的消息记录，是我们彼此都不愿承认的秘密。"

**同事型**:
- "工位之间3米的距离，我们用{count}条消息走了{days}天。"

## 2. 性格画像分析

### 分析维度

根据以下统计特征推断性格类型:

| 性格类型 | 判断依据 | 描述 |
|---|---|---|
| **讨好型** | 主动发起率>70%、对方回复慢仍持续、消息长度远超对方 | 总是先开口，用热情填满沉默。对方的每个句号，都被解读成省略号。 |
| **直球型** | 情感词频高、消息直接表达感受、不藏着掖着 | 喜欢就说喜欢，想你就发"想你"—— 简单粗暴，但真诚得让人招架不住。 |
| **傲娇型** | 回复简短但频率稳定、不主动但秒回、偶尔冷场但不消失 | 嘴上说着"随便"，手指却从未离开过对话框。 |
| **幽默型** | 表情/哈哈哈词频高、段子多、制造话题能力强 | 聊天界的氛围组组长，评论区永远的MVP。 |
| **佛系型** | 回复时差大、消息简短、偶尔失踪再出现 | 回复像东北的暖气——你永远猜不到什么时候来，但来了就很暖。 |
| **分析型** | 长文本多、逻辑性强、喜欢解释和分析 | 每条消息都像小论文，连"晚安"都要加个论据。 |

### 综合画像输出格式

```
🎭 {名字}的性格画像:
【核心类型】: {类型名} ({一句话描述})
【聊天风格】: {基于消息长度、回复速度、表情使用等描述}
【关系中的角色】: {在这段关系中扮演的角色}
```

## 3. 好感度 / 亲密度走势分析

### 分析信号

| 信号维度 | 积极信号 | 消极信号 |
|---|---|---|
| **消息量** | 月度递增或稳定高位 | 断崖式下降 |
| **回复速度** | 越来越快 | 越来越慢 |
| **主动性** | 双方交替主动 | 单方面维持 |
| **消息长度** | 对方的消息越来越长 | 越来越敷衍 ("嗯"、"好"、"哦") |
| **聊天时段** | 夜间聊天增加 | 仅限工作时间 |
| **内容深度** | 分享个人情感/秘密 | 仅聊表面话题 |

### 走势解读模板

- **上升期**: "从月度数据来看，你们的互动正处于升温阶段..."
- **蜜月期**: "这段时间的数据简直甜到齁..."
- **平稳期**: "日久见真情型——频率稳定，说明..."
- **冷却期**: "数据出现了一些值得关注的变化..."
- **单方面**: "数据显示了一些不对称性，需要正视..."

## 4. 追求/关系行动建议

### 建议原则
1. **基于数据**: 每条建议都要有数据支撑
2. **具体可执行**: "下周三晚上9点发条消息" 而非 "多联系"
3. **诚实温柔**: 如果数据不乐观，温柔地说出真相
4. **不鼓励纠缠**: 如果对方明显不感兴趣，建议放手

### 建议模板

**有戏型** (好感度上升):
- 基于对方活跃时段: "TA最活跃的时间是{hour}点，试试在这个时段发起话题"
- 基于共同高频词: "你们都爱聊{topic}，可以用这个做为约会切入点"
- 基于回复速度: "TA给你的秒回率{rate}%——这已经超过了'普通朋友'的基准线"

**需努力型** (不对称但有空间):
- "目前你的主动发起率({rate}%)偏高，试试策略性地减少主动，制造'被需要感'"
- "TA的回复较短，可能是习惯，也可能是还没完全打开心扉。试试抛出有趣的开放式问题"

**建议放手型** (长期单方面):
- "数据显示近{N}个月，对话几乎都由你发起，TA的回复也越来越简短。有时候放手不是放弃，是给自己一个更好的可能。"

## 5. 分析语言风格

- **专业但接地气**: 用数据说话，但不要像论文
- **有温度**: 分析的对象是感情，不要冷冰冰
- **有网感**: 适当使用年轻人的表达方式
- **emoji 适度**: 用来增加可读性，不要泛滥
- **金句点睛**: 每个模块结尾用一句精炼的话收束
